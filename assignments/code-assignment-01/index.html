Code Assignment 01

Create a web page and host it online

Create a web page as the first step in building your portfolio site. The portfolio will contain screenshots, descriptions, and links for each assignment you complete in this course. This page will be built up and improved throughout the semester as we cover new aspects of web development.

Process

  1. Start with the basic skeleton of a web page. Example: [demo] [code].
  2. “Brand” the page with a title, heading, logo, or other customizations to make it yours. Every assignment in this course will have a requirement like this, so save time and effort by coming up with a logo, color scheme, etc. to re-use on future projects.
  3. Create div elements to divide your content into sections.
  4. Add content (text, images, links) within the appropriate elements.
  5. Host your page online. Using Github, you can create a new repository named username.github.io (where username is your account’s username). Upload your HTML document to that repository. Your portfolio will then be accessible at https://username.github.io.

Graded Requirements

  1. A web page with valid HTML, hosted online
  2. Your own text, an image, and a link created with appropriate HTML elements
  3. Your own page title (within <title></title> tags)
  4. Proper indentation
