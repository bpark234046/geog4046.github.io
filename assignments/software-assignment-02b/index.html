Software Assignment 02b

Installing and configuring Apache on Linux

Set up Apache on Linux to serve a copy of your portfolio site. Apache is an open-source web server on Linux and macOS, with some support for Windows.

If you don’t already run your own Linux computer, you can install CentOS in a virtual machine on your computer, either Windows or macOS, or pay $0.007/hour for a server hosted online. In the steps below, use your hosted IP address instead of localhost, and submit the address on Moodle instead of a screenshot.

Process

These are the steps to install Apache on CentOS 7 using a local test server. The IP address of a hosted server can be used instead of localhost.

  1. Install and start the server:
    sudo yum install httpd
    sudo systemctl start httpd
  2. Copy your portfolio website’s files to the web root folder, /var/www/html/.
  3. Test http://localhost in a browser.
  4. Research more detailed instructions for installing Apache on CentOS or other distributions of Linux for things like automatically starting the web server after a reboot, and allowing public web traffic through the firewall.
