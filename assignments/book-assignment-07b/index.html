Book Assignment 07b

Publishing a geoprocessing service to ArcGIS Server

Complete Assignment 7B in Chapter 7 of GTKWebGIS, using the chapter tutorial and Q&A as guidance, to create a web app capable of processing spatial data using a geoprocessing service you published from ArcMap. You may use the book’s data and model, or you may create your own model to process your own data.

Process

Review the tutorial and assignment in the book for a detailed explanation of the process. Below is an overview of the steps involved.

  1. Gather the book files for Chapter 7, or find your own data.
  2. Edit the model from the book data, or create your own model in ArcMap’s ModelBuilder.
  3. Run the model and publish the results as a geoprocessing service.
  4. Create a web app using Web AppBuilder. You can re-use the map from Chapter 6 (earthquakes and hurricanes) or create a new one with your own data.
  5. Use the geoprocessing widget to connect to your published GP service.
  6. Customize, brand, share, and test the app.
  7. Take screenshots of your geoprocessing widget options before running the tool, and screenshot the results.

Graded Requirements

  1. An ArcGIS Online web app using the geoprocessing widget
  2. Two or more data layers that can be analyzed with the GP widget using your custom model, OR in the case of the book’s model and data, a layer that can be selected and downloaded as a zip.
  3. Customizations to give your app a professional look and feel
  4. Screenshots, description, and link to your web app on your portfolio website. Include screenshots of your GP widget’s options, and a screenshot of the results.
