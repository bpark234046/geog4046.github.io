Code Assignment 04

Introduction to JavaScript - Objects and arrays

Put values into an array, and create an object with the same values. In the console, log one of the values from the array, and then log the same value from the object.

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-04.js and include it with script tags in your portfolio page.
  2. In the JavaScript file, save some values of each type (number, string, boolean) to an array, and save the same values to an object.
  3. Create an array within the array and another within the object
  4. Choose one of the values to use as output to the console. Use console.log three times:
    • once to output your chosen value from the array,
    • once to output the same value from the object, and
    • once to output the same value using an alternative syntax.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. Scripts and styles in separate files
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title
    e. Proper indentation
    f. No errors in the console
  2. An array and an object containing values of at least three types: number, string, and boolean.
  3. An array within an array and an array within an object
  4. One of the values logged to the console three times:
    • from the array
    • from the object
    • again from the object but using a different syntax
