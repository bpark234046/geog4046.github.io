Code Assignment 07b

Libraries - jQuery

Include jQuery in your HTML document using a script tag, and use a jQuery event listener to run your script once the DOM has loaded.

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-07b.js and include it with script tags in your portfolio page.
  2. Include the jQuery JavaScript library using a src of https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js.
  3. Replace the JavaScript load event listener with the jQuery .ready() method to call your main/init function once the DOM has loaded.
  4. Add more jQuery listeners to do something simple when an event occurs, such as logging a message in the console when the user clicks on the page. See the list of jQuery mouse events, or the full list of events.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. Scripts and styles in separate files
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title
    e. Proper indentation
    f. No errors in the console
    g. All code inside a function/method called by an event listener when the page loads.
  2. jQuery included in the HTML document
  3. A jQuery even listener that begins running your code after the DOM has loaded
  4. At least two other jQuery event listeners that log a message to the console or perform some other task
