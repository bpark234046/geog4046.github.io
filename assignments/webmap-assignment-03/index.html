Webmap Assignment 03

Leaflet - Custom marker icons and styles

Apply some custom symbology to features on a Leaflet map. Change the icon used for the marker symbol, and change the colors of a polygon.

Process

This assignment is based on the Markers With Custom Icons tutorial, with the addition of polygon styling.

  1. Create a new JavaScript file webmap-assignment-03.js and include it with script tags in your map page. Re-use code from a previous assignment with a marker and polygon.
  2. Search online for an image to use as an icon. Any small PNG image with a transparent background should work. Optionally, create a shadow image. A shadow effect can be acheived in a graphics editor by skewing the icon image, changing the color or brightness/contrast to make the image solid gray, then applying a blur effect.
  3. Create an icon object in your JavaScript file. 
    var greenIcon = L.icon({
  iconUrl: 'leaf-green.png',
  shadowUrl: 'leaf-shadow.png',
  iconSize:     [38, 95], // size of the icon
  shadowSize:   [50, 64], // size of the shadow
  iconAnchor:   [22, 94], // point of the icon which will correspond to marker's location
  shadowAnchor: [4, 62],  // the same for the shadow
  popupAnchor:  [-3, -76] // point from which the popup should open relative to the iconAnchor
});
  4. Create a marker using the icon. For reference, compare the code of a marker without and with a custom icon.
    No custom icon: 
    let marker = L.marker([51.5, -0.09]).addTo(mymap)

    Custom icon:

    let marker = L.marker([51.5, -0.09], {icon: greenIcon}).addTo(map);
  5. Create an object containing some style options for the polygon. Refer to the Leaflet documentation for possible options to set, such as: 
    let polygonStyle = {
  color: 'orange',
  fillColor: 'yellow'
}
  6. Pass the polygonStyle object to the L.polygon method as the second argument. The first argument is an array of coordinates that form the polygon. 
    let polygon = L.polygon(polygonCoords, polygonStyle).addTo(mymap)

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML and CSS, hosted online
    b. Scripts and styles in separate files
    c. Your own page title
    d. Proper indentation
    e. No errors in the console
    f. A visible, functioning Leaflet map
  2. Your own icon, different from the tutorials.
  3. Non-default polygon options.
