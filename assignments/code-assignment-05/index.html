Code Assignment 05

Introduction to JavaScript - Loops and conditional statements

Use loops to access all of the elements in an array. Use conditional statements (if statements) to run some code only if a certain condition is met.

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-05.js and include it with script tags in your portfolio page.
  2. In the JavaScript file, create at least two objects with at least three properties and values each.
  3. Create an array and assign the objects you created above to elements of the array.
  4. Create a loop to step through each of the elements in the array (each element should be one of the objects you created).
  5. Within each loop, choose at least one object property to log to the console.
  6. Create a conditional statement that will only log the property to the console if its value meets a certain condition, e.g., is equal to true.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. Scripts and styles in separate files
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title
    e. Proper indentation
    f. No errors in the console
  2. An array containing at least two objects, which each contain at least three properties.
  3. A loop that steps through all of the values in the array.
  4. A conditional statement determining if a value gets logged to the console.
  5. An object property logged to the console, within a conditional statement, within a loop.
