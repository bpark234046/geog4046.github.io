Webmap Assignment 01

Leaflet - Quick start

Create a web map with Leaflet and display a basemap, point, and polygon with a popup. The assignment involves creating all of the necessary HTML, CSS, and JavaScript files and code to display a map on a webpage.

Process

This assignment is based on the Leaflet quickstart guide.

  1. In your portfolio repository, create a new file, map/index.html, with a basic HTML skeleton.
  2. In your map folder, create a new JavaScript file webmap-assignment-01.js and a CSS file map.css
  3. In your index.html page’s head and body, respectively add link and script tags to include your JS and CSS files.
  4. Also add link and script tags for Leaflet on the lines above your own CSS and JS: 
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.3.1/dist/leaflet.css">

    <script src="https://unpkg.com/leaflet@1.3.1/dist/leaflet.js"></script>

    Your HTML, CSS, and JavaScript files are now ready to have code added to them.

  5. Create a div element where the map will be drawn, using your own value for id. 
    <div id="mapid"></div>
  6. Create a CSS rule to give the map div a different height. 
    #mapid { height: 180px }
  7. Create the first line of JavaScript to create an empty map. Set your own variable name, map div id, coordinates, and zoom level: 
    //let someName = L.map('your-map-div-id').setView([yourLat, yourLon], yourZoom)
let mymap = L.map('mapid').setView([51.505, -0.09], 13)

    Note that the L object and the methods chained after it, map and setView, are specific to Leaflet.

  8. Add a basemap layer to the map using your preferred basemap’s URL and your own map object’s name’. See this list of basemap providers. 
    L.tileLayer('https://cartodb-basemaps-{s}.global.ssl.fastly.net/dark_nolabels/{z}/{x}/{y}.png').addTo(mymap)
  9. Mark a point of interest on the map using Leaflet’s marker method. Use your own variable names and coordinates. 
    let marker = L.marker([51.5, -0.09]).addTo(mymap)

    Draw a polygon around an area of interest.

    let polygon = L.polygon([
  [51.509, -0.08],
  [51.503, -0.06],
  [51.51, -0.047]
]).addTo(mymap);
  10. Create popups for your marker and polygon. Note the backslash allowing the use of an apostrophe. 
    polygon.bindPopup('St. Katharine\'s and Wapping')
marker.bindPopup('London Bridge Station')
  11. Create an “event listener” that listens for a click on the map and then runs a function to log the clicked location to the console. 
    function logCurrentCoordinates (event) {
  console.log('You clicked the map at ' + event.latlng)
}
mymap.on('click', logCurrentCoordinates)

    Or written a shorter way:

    mymap.on('click', function (event) {
  console.log('You clicked the map at ' + event.latlng)
})
  12. Consult the Leaflet documentation (or Google) to add a “polyline” to the map. No popup needed.

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML and CSS, hosted online
    b. Scripts and styles in separate files
    c. Your own page title
    d. Proper indentation
    e. No errors in the console
  2. Load the Leaflet CSS and JavaScript files
  3. A map div with a height
  4. A map with a basemap and features (marker, polygon, and line)
  5. Popups on the marker and polygon features
  6. An event listener that logs clicked locations
