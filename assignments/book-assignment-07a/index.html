Book Assignment 07a

Analysis tools in ArcGIS Online

Complete Assignment 7A in Chapter 7 of GTKWebGIS, using the chapter tutorial and Q&A as guidance, to create a web app capable of processing spatial data using the analysis widget in ArcGIS Online’s Web AppBuilder. The assignment says to use data and analyses for finding a new home, but you may choose other topics instead, as long as the data can be analyzed with the tools available in the Web AppBuilder widget.

Process

Review the tutorial and assignment in the book for a detailed explanation of the process. Below is an overview of the steps involved.

  1. Create a point layer with the location of a potential new home, or find/create your own data.
  2. Add layers from Living Atlas, Open Data portals, or other sources (should be relevant to real estate or your chosen topic).
  3. Choose tools from the Analysis widget in Web AppBuilder to perform some kind of useful analysis of your layers.
  4. Customize, brand, share, and test the app.
  5. Take screenshots of your analysis widget options before running the tools, and screenshot the results.
  6. Add a link, screenshot, and description of the app in your portfolio

Graded Requirements

  1. An ArcGIS Online web app using the analysis widget
  2. Data layers that can be analyzed with the analysis widget
  3. Customizations to give your app a professional look and feel
  4. Take screenshots of your analysis widget’s options before running the tools, and a screenshot of the results, such as a table or a new layer drawn on the map.
  5. Screenshot, description, and link to your web app on your portfolio website.

Submit on Moodle

  1. The URL of your web app
  2. Upload screenshots of analysis widget’s options and results
