Book Assignment 07a

Analysis tools in ArcGIS Online

Complete Assignment 7A in Chapter 7 of GTKWebGIS, using the chapter tutorial and Q&A as guidance, to create a web app capable of processing spatial data using the analysis widget in ArcGIS Online’s Web AppBuilder. You may choose any theme and corresponding data, as long as the data can be analyzed with the tools available in the Web AppBuilder “Analyze” widget.

Process

Review the tutorial and assignment in the book for a detailed explanation of the process. Below is an overview of the steps involved.

  1. Add hosted feature layers to a map/app, using data that will work with the Analyze widget in Web AppBuilder. Add related data from Living Atlas, Open Data portals, or other sources.
  2. Choose tools from the Analysis widget in Web AppBuilder to perform some kind of useful analysis of your layers. Portal users: see the Tutorial Notes for a list of analyses that work in the class Portal.
  3. Take screenshots of your analysis widget options before running the tools, and screenshot the output.
     
  4. Customize, brand, share, and test the app.

Graded Requirements

  1. An ArcGIS Online web app using the analysis widget
  2. At least one data layer that can be analyzed with the analysis widget’s tools
  3. Customizations to give your app a professional look and feel
  4. Take screenshot(s) of your analysis widget’s options before running the tools, and a screenshot of the output, such as a table or a new layer drawn on the map.
