Code Assignment 07a

Introduction to JavaScript - Libraries

Move a function into a separate .js file and include it in the HTML document with a script tag. Also include a third-party library, jQuery.

Process

These steps assume you will copy a function from the previous assignment, but the process can be applied to a class instead of function.

  1. Create a new JavaScript file code-assignment-07a.js and include it with script tags in your portfolio page.
    <script src="code-assignment-07a.js"></script>
  2. Create another JavaScript file using a name you choose for your library, such as FruitFunctions.js.
  3. Cut and paste your function code from the last assignment into the new library file.
  4. Include the library file in your portfolio page by using the script tag.
    <script src="FruitFunctions.js"></script>
  5. Add another script tag to include a third-party JavaScript library as well, jQuery. Use a src attribute value of https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js.
