Code Assignment 07a

Introduction to JavaScript - Libraries

Move a class into a separate .js file and include it in the HTML document with a script tag. Also include a third-party library, such as jQuery.

Process

These steps assume you will copy a class from the previous assignment, but the process can be applied to functions instead of a class.

  1. Create a new JavaScript file code-assignment-07a.js and include it with script tags in your portfolio page.
  2. Create another JavaScript file using the class name as the file name, such as Project.js.
  3. Cut and paste your class code into the new class file.
  4. Include the class file in your portfolio page by using the script tag.
  5. Include a third-party JavaScript library as well. jQuery, for example, would have a src attribute value of https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. Scripts and styles in separate files
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title
    e. Proper indentation
    f. No errors in the console
    g. All code inside a function/method called by an event listener when the page loads.
  2. A separate file containing a class, included through the HTML document
  3. A third-party library included in the HTML document
