Code Assignment 02b

Add style to a web page with CSS

Further develop your portfolio page. Structure the content (images, text) using CSS Grid, such as having images and text side-by-side.

Process

  1. Add a div to your HTML page to act as the “grid container”. Give it an id or class attribute so CSS rules can be applied to it, specifically.
  2. Add div elements inside of the grid container to act as “grid items”, which will hold the content. Give each one a unique id or class attribute.
  3. Create CSS rules to define the grid’s rows and columns, applied to the grid container element:
    • grid container width and height
    • number of rows and columns
    • height of rows, width of columns
    • size of gaps between rows and columns (if needed)
  4. Create CSS rules to define the position and size of the grid item elements, i.e., which “grid cells” the items should occupy.
  5. Move content (images, text) inside of the grid item elements.
  6. If needed, content can be further arranged within a grid cell by creating a new grid inside the cell’s elements.

Graded Requirements

  1. Requirements from the previous coding assignment:
    • A web page with valid HTML, hosted online
    • CSS rules in a separate file
    • A portfolio entry (a screenshot, description, and link) for each mapping assignment you’ve completed to date
    • Your own page title (within <title></title> tags)
    • Proper indentation
  2. A CSS grid with:
    • a grid container
    • at least 2 grid items
    • at least 2 columns or 2 rows
    • appropriate spacing between grid item content
    • content in the grid
