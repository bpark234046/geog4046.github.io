Software Assignment 01

Information Technology for Web GIS

Install and configure software tools for web pages and web maps. They will allow you to create, edit, test, debug, and publish web pages and maps.

Process

  1. Install Google Chrome to access ArcGIS Online and test websites you develop.
  2. Install Atom to view and edit text files (including CSV, GeoJSON, HTML, CSS, and JavaScript). Optionally, install add-ons under File > Settings > Install.
    • emmet - autocomplete for code
    • linter-js-standard - compliance checker for JavaScript Standard Style
    • minimap - show the entire contents of a document in a preview pane.
  3. Install Github Desktop to upload your websites from your computer to Github Pages, making your website available online. When you first start Github Desktop:
    • Create an account. Keep in mind that the username you choose will be used in the web address of your class portfolio website.
    • Create a new repository named yourusername.github.io. This will be your portfolio website.
    • Clicking Repository > Open in Atom will let you start creating and editing files to upload to the website.
    • After making changes in Atom, commit the changes in Github Desktop by typing a Summary and clicking Commit to master.
    • Upload the repository’s file to your Github Pages website by clicking Publish Repository.
    • The website you just published is now accessible at https://yourusername.github.io.
  4. Install ArcGIS Desktop student trial to publish maps to an ArcGIS Server. You can create an Esri account to sign in, and download and license the software using a license key provided by your instructor.
