Book Assignment 05

Build your own app with Web AppBuilder in ArcGIS Online

Create a new web app with ArcGIS Online Web AppBuilder using your own data, either a new dataset or a feature layer published in a previous assignment.

Process

Do not directly follow the assignments in chapter 4 and 5 for this exercise. Instead, refer to those chapters as a rough guide for completely the steps below.

  1. Find or create a data layer with multiple attributes that can be charted and queried (see 5.4).
  2. Create a web map with your data layer(s)
  3. Create a web app with Web AppBuilder, showing your web map
  4. Add and configure the chart, query, and print widgets
  5. Customize, brand, share, and test the app.
  6. Add a link, screenshot, and description of the app in your portfolio

Graded Requirements

  1. An ArcGIS Online web app created with Web AppBuilder
  2. Different data than that used in the tutorial
  3. Chart, query, and print widgets
  4. Customizations to give your app a professional look and feel
  5. A screenshot, description, and link to your web app on your portfolio website

Submit on Moodle

  1. The URL of your web app
