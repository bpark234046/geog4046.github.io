Code Assignment 06c

Introduction to JavaScript - Event listeners and more functions

Use “event listeners” to call functions to perform specific actions when an event, such as a mouse click, is triggered in the browser. Move all JavaScript code into functions that are called by event listeners (or called by other functions called by event listeners).

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-06c.js and include it with script tags in your portfolio page.
  2. Move all code into a function or class method, or multiple functions if the code performs multiple tasks. Create new functions/methods if necessary.
  3. Create event listeners that call the function/method(s) created in the previous step.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. Scripts and styles in separate files
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title
    e. Proper indentation
    f. No errors in the console
    g. A class
  2. At least one event listener that calls a function/method
  3. All code except the event listener(s) must be inside a function/method.
