Code Assignment 06a

Introduction to JavaScript - Functions

Use a function to create a block of code that performs a specific task and can be used and reused from anywhere in the script.

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-06a.js and include it with script tags in your portfolio page. Copy the code from the previous coding assignment.
  2. Declare a function:
    • Use a camelCase name that desribes the task, such as describeFruit.
    • Specify function arguments, e.g., function describeFruit (fruit) {}
  3. Select some code that performs a specific task and move it into the function, between the opening { and closing }, e.g., 
    let description = 'The ' + fruit + ' is a delicious snack.'
  4. Use return to let the function output a value, e.g., return description.
  5. Call the function from somewhere else in the script, with some value or variable as a parameter, e.g., describeFruit('apple'). Assign the functionâ€™s returned value to a variable and log it to the console, e.g., 
    let myfavfruit = 'apple'
let description = describeFruit(myfavfruit)
console.log(description)

    Which is the same as writing:

    let description = describeFruit('apple')
console.log(description)

    Which is also the same as writing:

    console.log(describeFruit('apple'))
