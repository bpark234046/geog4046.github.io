Webmap Assignment 05

Leaflet - Layer groups and controls

Use Leaflet’s layer groups and controls to add a tool to the map allowing users to switch to a different basemap.

Younger and older populations: median age by state, 2010

Process

This assignment is based on the Layer Groups and Layer Controls tutorial, but since we already have GeoJSON data displayed on the map from the previous assignment, we will skip to the Layer Controls section.

  1. Duplicate the previous assignment’s JavaScript file and rename it webmap-assignment-05.js. Update the script tag in your map page.
  2. In the JavaScript file, find the line in your code that creates the basemap object. Save it to a variable if it isn’t already. Since we will be adding an additional basemap, we need to save each one to a variable: 
    L.tileLayer(url).addTo(myMap)

    …becomes:

    let myLightmap1 = L.tileLayer(url).addTo(myMap)

    …where url1 is the web address of the basemap service you choose.

  3. Browse the list of basemap providers to find a second basemap and copy the code. 
    let myDarkmap2 = L.tileLayer(url2)

    The second basemap should not be added to the map yet, so remove the addTo method.

  4. Repeat step 3 to add a third basemap option.
  5. Leaflet has a method for adding a control to the map that lets the user show/hide basemaps. It is the L.control.layers method and it’s first argument is an object containing a list of basemaps. Create this object before creating the control. The object’s property names will appear on the map, so give them something short but descriptive: 
    let myBasemaps = {
  'Light basemap': myLightmap1,
  'Dark basemap': myDarkmap2
}
  6. Call the L.control.layers method with the basemap list object given for the argument. 
    L.control.layers(myBasemaps).addTo(myMap)
