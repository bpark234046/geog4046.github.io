Software Assignment 03

Installing and configuring ArcGIS Enterprise

Install and configure ArcGIS Enterprise, a GIS application that can serve maps and data as web services. Install ArcGIS Server to host the web services and ArcGIS Web Adaptor to act as a public gateway to those web services.

Process

These steps are for the preinstalled Windows Server VM, which you can access with the hostname, username, and password provided to you in class.

  1. Look for the ArcGIS Enterprise installation files in c:\install\.
  2. Install ArcGIS Server.
  3. Authorize ArcGIS Server.
    • The .prvc file will be provided in class.
    • At the Software Authorization Wizard, select the option, I have received an authorization file....
    • Browse to and select the .prvc file.
    • Leave the default information.
  4. Create a new site.
    • Choose a username and password that is long yet memorable.
  5. Install ArcGIS Web Adaptor.
    • The ArcGIS Server URL should be something like https://myserver.example.edu:6443.
    • Enter the same username and password you just used for ArcGIS Server.
    • Check Enable administrative access....
    • Click Configure.
  6. Enable HTTPS:
    • Open IIS Manager.
    • Click on the default site, click Bindings, Add..., select options Type: https, SSL certificate: WMSVC-SHA2.
  7. Test that the ArcGIS REST Services Directory site works. The URL should be something like:
    https://myserver.example.edu/arcgis/rest/services
  8. Log into ArcGIS Server Manager at:
    https://myserver.example.edu/arcgis/manager
  9. Under Services > Manage Services:
    • Delete the Sample World Cities service.
    • Start the Geometry service.

Graded Requirements

  1. An ArcGIS Server site
    a. Deleted Sample World Cities service
    b. Started Geometry service
  2. An ArcGIS Web Adaptor site in IIS
    a. HTTPS works
    b. Server Manager accessible through web adaptor
