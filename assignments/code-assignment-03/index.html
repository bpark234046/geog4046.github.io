Code Assignment 03

Introduction to JavaScript - Variables and the console

Add JavaScript to your portfolio site to print the values of variables to the browser’s developer console.

Process

  1. Create a new JavaScript file code-assignment-03.js in the same folder as your portfolio HTML file.
  2. Use a script tag in your portfolio page’s HTML to load the JavaScript file.
  3. In the JavaScript file, create variables of different types: number, string (text), boolean (true/false), and array.
  4. Use console.log to print a message to the browser’s developer console, using your variables as input. Add a descriptive label to the output, e.g., Title: Map of Recent Earthquakes instead of just Map of Recent Earthquakes.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. CSS rules in a separate file
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title (within <title></title> tags)
    e. Proper indentation
  2. JavaScript included in a separate file.
  3. At least four variables, one of each type: number, string, boolean, and array.
  4. Display the values of each variable in the browser’s developer console. For each value, combine the variable value output with some text to explain the output
  5. No errors in the console.
