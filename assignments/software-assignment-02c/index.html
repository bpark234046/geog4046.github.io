Software Assignment 02c

Installing and configuring IIS in Windows Server 2016

Set up IIS, the web server included with Windows, to run a copy of your portfolio site on a test server.

Process

  1. See the steps for adding the Web Server role to Windows Server as described in the ArcGIS Server documentation.
  2. Copy the portfolio website’s files to the IIS website’s “root directory”, which by default is c:\inetpub\wwwroot\.

Graded Requirements

  1. Screenshot of the web browser loading your portfolio at http://localhost.
  2. Screenshot of IIS Manager on your server.

  1. Upload the screenshots.
