Code Assignment 06b

Introduction to JavaScript - Classes

Create your own blueprint for an object by defining a class. Replace variables and functions with properties and methods within the class.

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-06b.js and include it with script tags in your portfolio page.
  2. Create a class with:
    • a constructor that saves an array of data to a property of the class, e.g., this.data
    • methods containing code that was previously in a function
  3. Outside of the class, create a variable, e.g., portfolio, to hold a new Project object.
  4. Create a for loop that can walk through the array and call a method on each item in the array. Using the above examples for the constructor and portfolio variable, the array would be portfolio.data and each array item would be accessed in the loop as portfolio.data[i]. Methods would be called as portfolio.yourMethodName(), and could accept the current array item as an argument, like portfolio.yourMethodName(portfolio.data[i]).
  5. Log the result of the method to the console.

Graded Requirements

  1. Requirements from previous coding assignments:
    a. A web page with valid HTML, hosted online
    b. Scripts and styles in separate files
    c. A portfolio entry for each mapping assignment to date
    d. Your own page title
    e. Proper indentation
    f. No errors in the console
  2. A class with:
    a. a constructor
    b. at least one method
    c. at least one property containing an array
  3. A variable representing a new instance of the class
  4. A loop that:
    a. loops through each array item in a class property
    b. calls a method of the class on each item in the array
  5. Console output showing the result of the method.
