Code Assignment 07c

Libraries - jQuery DOM interaction

Modify the script from Code Assignment 10 to use jQuery methods to interact with the DOM.

Process

These steps assume you will copy code from the previous assignment and build upon it in a new .js file to add features to your portfolio.

  1. Create a new JavaScript file code-assignment-07c.js and include it with script tags in your portfolio page.
  2. Copy and paste the JavaScript from Code Assignment 10 into the new file, but remove the class declaration since we are including the class as a library from our HTML with a script tag.
  3. Duplicate and rename your class file from Code Assignment 11, e.g., change Project.js to jQueryProject.js. Change the src value of the HTML script tag to match.

  4. Begin modifying the JavaScript in the class file to use jQuery to modify the DOM. See examples in the table below.

    JavaScript jQuery
    document.getElementById('id') $('#id')
    someHtml.getElementsByTagName('img') $('img', someHtml)
    cloneNode(true) clone(true)
    someElementArray[0] someElementArray.eq(0)
    someElementArray[1] someElementArray.eq(1)
    someElement.setAttribute('href', someUrl) someElement.attr('href', someUrl)
    someElement.innerHTML = 'some text' someElement.html('some text')
    someParentElement.appendChild(someHtml) someParentElement.append(someHtml)

     
    An example conversion for a class method that clones an HTML template using vanilla JavaScript:

    cloneProjectHtml (templateId, newId) {
  let template = document.getElementById(templateId)
  let projectHtml = template.cloneNode(true)
  projectHtml.setAttribute('id', newId)
  return projectHtml
}

    becomes this with jQuery:

    cloneProjectHtml (templateId, newId) {
  let template = $('#' + templateId)
  let projectHtml = template.clone(true)
  projectHtml.attr('id', newId)
  return projectHtml
}
return $('#' + templateId).clone(true).attr('id', newId)

    Or it could all be written as a single line:

    cloneProjectHtml (templateId, newId) {
  return $('#' + templateId).clone(true).attr('id', newId)
}
