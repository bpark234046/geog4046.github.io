Book Assignment 10

Programming with the ArcGIS JavaScript API

This assignment is based on Assignment 10 in Chapter 10 of GTKWebGIS, using the chapter tutorial and Q&A as guidance. Create a web page that displays two views (2D and 3D) of the spatial dataset of your choice. Use a dataset not used in the textbook, though you can use datasets from previous assignments if they are not from the book. Set a layer’s symbol size and/or color based on data in one or more fields (see pg 382, step 5; choose a dataset with that in mind).

Process

Use examples from the tutorial to create the page, copying code and changing variable values where needed. The HTML file names below refer to the textbook files used in the chapter 10 tutorial. Start by creating a file on your computer named book-assignment-10.html.

  1. Choose a dataset that has attributes on which to base the layer symbology, and get the textbook’s code sample files for chapter 10.
  2. Copy all code from 2d3d_Views_linked_key.html into your assignment6.html file.
  3. Copy appropriate 3D map/scene code from polygon_extrusion_key.html, replacing code where needed.
  4. Add necessary modules, delete unused modules.
  5. Change the variable name of the 3D view object; don’t use var twice on the same variable.
  6. Change div ID when creating the 3D view.
  7. Change viewpoint/camera positions for 2D and 3D views.
  8. Set a layer’s extrusion size and/or color based on data in one or more fields (ch 10, pg 382, step 5).
  9. Change the page title to describe your maps.
  10. Test, debug, and fix.
  11. Upload book-assignment-10.html to Github or other web host

Graded Requirements

  1. Two maps/views visible on the same page
  2. Non-textbook data visible in both views
  3. Customized 3D layer symbols
  4. Appropriate starting viewpoint in each view
  5. Navigation (pan, zoom) linked from one view to the other
  6. Relevant page title
  7. No errors in console (although “favicon” errors are ok)
