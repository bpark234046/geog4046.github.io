Book Assignment 09

3D GIS with ArcGIS Online

This assignment using ArcGIS Online Scene Viewer will not follow Chapter 9 in Getting to Know Web GIS exactly, but the chapter can be referred to for additional information about scenes, ArcGIS Pro, and 3D layers.

Process

Review the tutorial and assignment in the book for a detailed explanation of using the ArcGIS Online scene viewer. Below is an overview of the steps involved.

  1. Create a new scene my clicking the Scene tab after logging into ArcGIS Online, or in My Content by clicking Create > Scene.
  2. In the scene viewer, you can add layers by searching the ArcGIS Online gallery or entering a layer URL, which could be a REST URL (ending in ...MapServer/0, for example).
  3. Customize the 3D symbology based on an attribute value, such as taller markers for higher values
  4. Save the scene and share it to create a web app

Graded Requirements

  1. A data layer in a web scene in a web app
  2. Layer symbology customized based on an attribute value
  3. Customizations to give your app a professional look and feel
