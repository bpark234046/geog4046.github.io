Process

Review the tutorial and assignment in the book for a detailed explanation of the process. Below is an overview of the steps involved.

  1. Obtain data, such as earthquakes from the USGS.
  2. Add the data to a scene in ArcGIS Pro
  3. Customize the symbology based on an attribute value, such as darker colors for higher values
  4. Publish the scene to ArcGIS Online
  5. Create an app to view the scene
  6. Add a screenshot, and description of the app in your portfolio

Graded Requirements

  1. A data layer in a web scene in a web app
  2. Layer symbology customized based on an attribute value
  3. Customizations to give your app a professional look and feel
  4. A screenshot and description of your app on your portfolio website
