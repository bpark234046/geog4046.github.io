Book Assignment 06b

Publishing a web service to ArcGIS Server

Complete Assignment 6B in Chapter 6 of GTKWebGIS, using the chapter tutorial and Q&A as guidance, to publish your own data as a map service from ArcMap to our class ArcGIS Server, and then add the service to a web map/app.

Process

Review the tutorial and assignment in the book for a detailed explanation of the process. Below is an overview of the steps involved.

  1. Connect to the class GIS server using the connection file provided on Moodle (skip GTKWebGIS section 6.1).
  2. Create a map using your own data, setting appropriate symbology.
  3. Publish your map as a service in your own folder.
  4. Add your map service to a new map in ArcGIS Online.
  5. Enable popups showing relevant attribute data.
  6. Customize, brand, share, and test the app.
  7. Add a link, screenshot, and description of the app in your portfolio

Graded Requirements

  1. An ArcGIS Online web app showing the map service you published from ArcMap
  2. Customizations to give your app a professional look and feel
  3. A screenshot, description, and link to your web app on your portfolio website

Submit on Moodle

  1. The URL of your web app
  2. The REST URL of your map service
