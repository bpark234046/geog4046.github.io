Code Assignment 2

Add style to a web page with CSS

Further develop your portfolio page. In addition to adding a screenshot, description, and link for your completed assignment(s), add some style using CSS, such as font size, text alignment, background color, etc.

Process

  1. Create a new stylesheet file (e.g., style.css) in the same folder as your HTML file.
  2. Add a <link> element in your HTML to include the stylesheet.
  3. Add rules to your CSS file to style certain elements in your HTML. Where appropriate, add classes to HTML elements so the rules get applied to them.

Graded Requirements

  1. A web page with valid HTML linking a valid CSS file, hosted online
  2. At least three of your own CSS rules applied to HTML elements
  3. For those three or more rules, have at least one rule for an HTML element, one rule for an id, and one rule for a class
