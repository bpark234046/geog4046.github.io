Assignment Selection

This is a list of all possible assignments including required core assignments and electives. Complete the core set of assignments and choose five electives.

Every core assignment needs to be completed. Submitting core assignments can unlock the subsequent core assignments as well as certain electives.

Submit these in the order assigned on the class schedule: starting with Level 1 assignments and finishing with Level 10.

Electives

Submit at least five elective assignments after the prerequisite assignments have been submitted. The five electives can be spread across all or some concentrations, or put into one. The concentrations are groups of electives that loosely follow four themes: GIS, web design, system administration, and software development.

Start with Level 1 electives.

Concentrations

GIS
Assignments may be focused more on data and cartography.

Web Design
Assignments are more about creating the layout and style of web pages

System Administration
Assignments may involve installing and configuring software involved in web GIS

Software Development
Assignments require progressively advanced programming
