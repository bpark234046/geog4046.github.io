Book Assignment 02

Layer symbology and popups in ArcGIS Online

Complete Assignment 2 in Chapter 2 of GTKWebGIS, using the chapter tutorial and Q&A as guidance, to create a map with your choice of data and symbology.

Process

Review the tutorial and assignment in the book for a detailed explanation of the process. Below is an overview of the steps involved.

  1. Find and prepare the data of your choice
  2. Create a web map and add the data
  3. Style the layer with symbology based on data values
  4. Configure a popup to display data, charts, or media when a feature is clicked
  5. Add a Living Atlas layer related to your data
  6. Create the web app from the map
  7. Configure the app
  8. Share the app
  9. Test the app’s accessibility in incognito/private browsing mode
  10. Add a link, screenshot, and description of the app in your portfolio

Graded Requirements

  1. An ArcGIS Online web app displaying your data layer
  2. Symbology based on the layer’s attributes, e.g., larger icon for greater values
  3. Popup showing layer attributes and at least one image, video, link, chart or other media unique to the feature being clicked
  4. Customizations to give your app a professional look and feel
  5. A screenshot, description, and link to your web app on your portfolio website

Submit on Moodle

  1. The URL of your web app
