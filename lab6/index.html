Create a web page that displays two views, 2D and 3D, of the spatial dataset of your choice. Use a dataset not used in the textbook, though you can use datasets from previous assignments if they were not from the book. Process

Use examples from the tutorial to create the page, copying code and changing variable values where needed. The HTML file names below refer to the textbook files used in the chapter 10 tutorial. Start by creating a file on your computer named assignment6.html.

  1. Copy all code from 2d3d_Views_linked_key.html into assignment6.html
  2. Copy appropriate 3D map/scene code from polygon_extrusion_key.html, replacing code where needed
  3. Add necessary modules, delete unused modules
  4. Change the variable name of the 3D view object; don't use var twice on the same variable
  5. Change div ID when creating the 3D view
  6. Change viewpoint/camera positions for 2D and 3D views
  7. Set a layer's extrusion size and/or color based on data in one or more fields (ch 10, pg 282, step 5)
  8. Change the page title to describe your maps
  9. Test, debug, and fix
  10. Copy assignment6.html to your class web folder, Github Pages, or other web host
  11. Submit the link on Moodle

Graded Requirements

  • Two maps/views visible on the same page
  • Non-textbook data visible in both views
  • Customized 3D layer symbols
  • Appropriate starting viewpoint in each view
  • Navigation linked from one view to the other
  • Page title
  • No errors in console (although "favicon" errors are ok)
