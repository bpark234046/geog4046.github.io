Assignment 09: Create a Leaflet map

Create an interactive map using the Leaflet JavaScript library. Refer to the Leaflet documentation and examples provided in class. Use Leaflet functions to draw a shape on the map, and use jQuery and Leaflet to display a GeoJSON data layer.

Process

Start with a basic HTML skeleton and supporting CSS and JavaScript files. Copy and modify JavaScript code from the Leaflet documentation and class examples. Make sure your HTML includes a box (div) in which to draw the map, and make sure the box has a height defined in your CSS file.

  1. Create files and folders
  2. Start with a basic HTML template [Leaflet getting started, GEOG4046 HTML skeleton]
  3. Follow the Leaflet quick start guide, referring to the steps below for guidance. You will modify the feature coordinates later.
  4. Under "Setting up the map" you can use the Openstreetmap basemap instead of Mapbox Streets so you don't need an account and API key from Mapbox. [example]
  5. You can look through this list of basemap providers and copy the example code into your script. ArcGIS map services can also be used if their WMS capability has been enabled.
  6. Find data in GeoJSON format to use in the next steps. [Baton Rouge National Landmarks, USGS earthquake data, other sources on ArcGIS Opendata]
  7. Add a GeoJSON layer to the map. Use jQuery's getJSON function to fetch the data [example]. Then use Leaflet's geoJSON function to turn the data into a layer. See the Leaflet documentation on GeoJSON—specifically, look at the first line of code in the section "The GeoJSON layer".
  8. Add a popup to the GeoJSON features. You have to create a function that adds a popup. Then use the onEachFeature option of the geoJSON method to call the function. [example]
  9. Modify the shapes created earlier to be around the same location as your GeoJSON layer. Keep at least one shape, and change it to make sense with your GeoJSON layer, such as drawing a circle around a group of related GeoJSON features.

Graded Requirements

  • Include jQuery JavaScript
  • Include Leaflet CSS
  • Include Leaflet JavaScript
  • Your own CSS
  • Your own JavaScript
  • Scripts and styles in separate files
  • Your own page title
  • A Leaflet map
  • A basemap
  • A GeoJSON layer
  • Shape(s) drawn with Leaflet. Could be a marker, line, or polygon. It should complement or makes sense with your GeoJSON data
  • Popups on the GeoJSON layer
  • Proper indentation
  • Commented JavaScript code
  • No errors in developer console
