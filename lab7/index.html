Assignment 07: Create a website with HTML, CSS, and JavaScript

Create a website. Use the Bootstrap grid system to make at least one row and two columns. Collect input from the user and display the input on the page. Use JavaScript to automatically generate the copyright year in the footer.

Process

Use samples of code to construct a website.

  1. Refer to examples given in class or examples from the web to start with the basic skeleton of a web page.
  2. Put styles and scripts into separate files.
  3. Include those JavaScript and CSS files in your HTML page with the script and link tags.
  4. Create div elements to lay out the Bootstrap grid structure of the page. [example]
  5. Add your content (e.g., text, images) within the appropriate elements.
  6. Use an input element to get some data from the user. [example]
  7. Add a button the user can click to submit their input.
  8. Create an empty HTML element (e.g., a div) where your JavaScript can print the user's input.
  9. Use samples of code to build a script that reacts to the user clicking a button by reading the input and printing it on the page [example]
  10. Create an empty HTML element (e.g., a span) to hold the year value generated by your JavaScript. [example]
