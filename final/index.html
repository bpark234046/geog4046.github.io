Final Project: JavaScript Web App

Use Leaflet and the ArcGIS JavaScript API to create an application displaying interactive maps. Each map should have layers that make sense, and the application should have some controls that change the map(s) or map view(s).

Process

You will need to create an HTML page to hold the maps and interface elements; CSS to style the HTML, especially the map containers; and JavaScript to display the maps and layers, and handle user input.

  1. Create files and folders
  2. Start with a basic HTML template [basic Leaflet, GEOG4046 HTML skeleton] that links all necessary libraries: CSS and JavaScript for Leaflet, ArcGIS, jQuery, and Bootstrap.
  3. Add HTML and CSS to create interface elements, such as a navigation bar, images, a header, a footer, boxes of text, or buttons.
  4. Give the app a "brand"—consistent styling, organization name, and a generally professional appearance that makes the page look like a functional product and not a technical demo.
  5. Add a Leaflet map to your page, or create a separate page linked from your home page. See assignment 9 or the Leaflet quick start example to create a basic map.
    • Add a basemap (tile layer), covered in the quick start guide. See this list of providers for additional basemaps.
    • Add GeoJSON layers. Create a popup, modal, labels, or some other display of information for each feature. Apply custom styles or markers where needed to create a readable, informative map.
    • Add controls to show/hide layers. You can use Leaflet controls or make your own buttons in HTML with similar functionality. For example, create a button in HTML, listen for clicks on it with jQuery, and run a Leaflet function to toggle a layer when clicked.
    • Use a plugin to add extra functionality to Leaflet. See the list of plugins.
  6. Add some JavaScript to create a map with the ArcGIS API. ArcGIS and Leaflet JavaScript can coexist in the same file or be separated. You could use either the 2D or 3D code from the 2d3d_Views_key.html textbook example. You can use existing data sources or publish new ones to a free ArcGIS Online account or the class GIS server. See GTKWebGIS chapters 6 (publishing), 9 (creating 3D scenes), and 10 (JavaScript API).
  7. Collect user input, for example using a text box, checkbox, or select dropdown. Use JavaScript to respond to the user's input, such as manipulating the page or map.
